T3defaults

My TYPO3 Defaults

Download this project as a .zip file Download this project as a tar.gz file

Complete TYPO3 TS config/constant definition for all things i need. (TYPO3 version 6.2.4)

Including the following

  • Typoscript constants and setup for the most important things
  • Page config
  • Page types
  • Navigations(Header,Main,Left,Breadcrumb)
  • Objekts(Logo,Claim,Search...)
  • CSS
  • JS
  • Complete fluid based
  • Responsive
  • Grid generator (max_width,max_cols,col_width,col_margin,unity) see constants
  • Sass file(normal.scss) for template css. You need compass to compile the file or you must use the generated normal.css
  • Less file(news-basic.less) for news extension css. You need less to compile the file or you must use the generated news-basic.css
  • Extensions(news,fluid,direct_mail_subscription,indexed_search...)
  • Debug
  • Debugbar(Press STRG+ALT+SHIFT+D in FE)
  • Test touch with url parameter &touch=1
  • Webanalytic(E-Tracker and google analytics)
  • and more...

Installation

  • Save it to your fileadmin (Use same folder structure)
  • Create a root TS template in BE
  • Copy/paste basis_constants.ts into your constant field
  • Copy/paste into your setup field
  • Include css_styled_content into your static template section
  • Save
  • Switch to template view -> constant editor
  • Select T3D_Config
  • Enter your base url
  • Save and view it in FE
