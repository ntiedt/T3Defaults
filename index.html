My TYPO3 Defaults

Download this project as a .zip file Download this project as a tar.gz file

Complete TYPO3 TS config/constant definition for all things i need.

Including the following

  • Typoscript constants and setup for the most important things
  • Page config
  • Page types
  • Navigations(Header,Main,Left,Breadcrumb)
  • Objekts(Logo,Claim,Search...)
  • CSS
  • JS
  • Extensions(tt_news,fluid,templavoila,indexed_search...)
  • Debug
  • Webanalytic
  • and more...

Installation

  • Save it to your fileadmin (Use same folder structure)
  • Create a root TS template in BE
  • Copy/paste basis_constants.ts into your constant field
  • Copy/paste into your setup field
  • Include css_styled_content into your static template section
  • Save
  • Switch to template view -> constant editor
  • Select T3D_Config
  • Enter your base url
  • Save and view it in FE
